# 무한매수법 V4.0 — 쉽게 읽는 설명서

> 같은 폴더의 `strategy-v4-spec.md` 는 코드 짤 때 보는 **개발용 정밀 스펙**.
> 이 문서는 사람이 편하게 읽으며 전략을 "감으로" 이해하는 **쉬운 설명서**.
>
> ⚠️ 라오어 원문은 외부 재배포 금지. 이 문서는 개인 학습/개발용 내부 정리.

## 한 줄 요약
3배 레버리지 ETF(TQQQ·SOXL)를 **매일 조금씩 나눠 사고, 평균단가보다 조금 오르면 나눠 판다.** 다 팔리면(보유 0) 처음부터 다시. 이걸 무한 반복해서 "무한매수".

## 왜 이렇게 하나?
3배 ETF는 출렁임이 심해서 한 번에 크게 사두면 오래 물릴 때 크게 다친다.
그래서 **한 방에 안 사고**, 원금을 20~40조각으로 쪼개 매일 한 조각씩 산다.
떨어지면 더 싸게 더 사서 평단을 낮추고, 평단보다 살짝 오르면 조금씩 판다.
→ "조금씩 사서 조금씩 익절" 을 기계적으로 반복하는 자동 시스템.

## 핵심 용어 3개만 알면 됨
- **T값**: "며칠째 사고 있나"를 나타내는 진행 눈금. 한 번 다 사면 +1, 절반만 사면 +0.5, 조금 팔면(쿼터매도) ×0.75로 줄어듦.
- **별지점(★)**: 그날의 **기준선 가격**. 이 선 위면 팔고, 아래면 산다. `평단 × (1+별%)`로 매일 새로 계산. 처음엔 평단 위에 있다가, 후반부로 갈수록 평단 아래로 내려옴(=손해 봐도 정리 시작).
- **분할수(20/40)**: 원금을 몇 조각으로 나눌지. 조각 많을수록(40) 천천히·안전하게, 적을수록(20) 빠르고·공격적.

## 하루 루틴 (일반모드)
매일 장 마감 기준으로 자동 주문을 깐다:
1. **산다**: 그날 살 돈(=잔금÷남은회차)을 별지점·평단 근처에 LOC로 깔고, 혹시 폭락하면 더 줍게 그 아래로 1주씩 사다리 주문도 깐다.
   - 전반전(T가 절반 전): 절반은 별지점, 절반은 평단에서 산다.
   - 후반전(T가 절반 넘음): 전액 별지점에서 산다.
2. **판다**: 가진 것의 **1/4은 별지점에서**, **나머지 3/4은 평단보다 15%(TQQQ)/20%(SOXL) 위에서** 판다.
3. **다 팔려서 보유 0 → 한 사이클 끝.** 다음날 처음부터 다시 시작.

> 비유: 별지점은 매일 움직이는 "물 높이 표시선". 가격이 선 위로 오면 한 바가지 퍼내고(매도), 선 아래로 빠지면 한 바가지 더 붓는다(매수).

## 쉬운 예 (느낌만)
- 원금 2만달러, 40분할 → 하루 약 500달러어치씩 산다.
- 사다 보니 평단 60달러. 별% 계산해보니 별지점 59.55달러.
  - 가진 것의 1/4은 59.55에 팔 주문, 나머지는 69(=평단+15%)에 팔 주문.
  - 동시에 59.54 아래로 살 주문도 깔아둠 → 더 떨어지면 더 싸게 줍는다.
- 운 좋게 다 팔리면 끝. 수익 챙기고 처음부터 반복.

## 일이 잘 안 풀릴 때 — 리버스모드(소진모드)
계속 떨어져서 **원금을 거의 다 써버리면**(T가 거의 분할수에 도달) "리버스모드"로 바뀐다.
- 이제는 **더 살 돈이 거의 없으니, 주로 판다.** 첫날은 가진 것의 일부를 무조건(시장가) 팔고, 다음날부터 조금씩(가진 것의 1/N 등분) 별지점 위에서 판다.
- 남은 푼돈 + 판 돈을 모아 1/4씩만 조심스럽게 산다(쿼터매수).
- **가격이 평단 근처(평단의 −15%/−20% 선)까지 회복되면** 리버스 끝 → 다시 일반모드로 복귀. (T값은 이어서 사용)
- 핵심: 손해 구간에서도 **규칙대로 천천히 정리**하며 버틴다. 외부 돈 추가 투입은 절대 안 함.

## 마음가짐 / 규칙
- 정한 원금 **안에서만** 굴린다. 중간에 돈 더 안 넣는다.
- 손절 구간이 와도 규칙 유지 — 규칙 어기면 그건 무한매수법이 아니라 그냥 변형.
- 단리(원금만 재사용)가 기본, 복리(수익도 재투입)는 선택.

## 자주 헷갈리는 점
- "별% 가 마이너스?" → 후반전엔 정상. 평단보다 낮은 가격에 팔며 평단·물량을 정리하는 구간.
- "왜 1주씩 사다리?" → 갑자기 크게 빠지는 날 더 싸게 줍기 위한 보험.
- "LOC가 뭐야?" → 종가 기준 지정가 주문. 장 안 보고 종가로만 체결돼 충동매매를 막아줌.

# 무한매수법 V4.0 백테스트 리포트 (3년)

- 기간: **2023-05-30 ~ 2026-06-05** (약 3.0년, 758거래일)
- 원금: **$10,000 / 종목**, **40분할**
- 데이터: yfinance 일봉(분할·배당 조정)
- 엔진: `strategy.py`(코어) + `backtest.py`(체결 시뮬). 인과 보장(d일 주문은 d−1 종가/5일평균 기준, d일 OHLC 체결).
- 변형: **라오어순정**(단리·별%15/20·큰수=평단·리버스=별지점) vs **언이변형**(복리·목표수익률·큰수=종가·리버스=5일평균).

## 해석 — 정직하게

1. **강세장에선 Buy&Hold에 크게 뒤진다 (설계상 당연).** 무한매수는 상승분을 일찍 익절하고 현금을 들고 있어 폭등장 수익을 다 못 먹는다. 대신 **MDD가 훨씬 낮다**(3x ETF 단순보유 MDD는 보통 60~80%, 여기선 20~42%). = 변동성/위험을 깎는 전략이지 수익 극대화 전략 아님.

2. **사이클 승률 100% 는 착시.** 사이클은 *전량 익절*돼야 종료되므로 닫힌 사이클은 무조건 +. **손실은 미청산·리버스 포지션에 숨는다.** 이 기간(2023~2026 강세장)은 리버스 진입이 **0~2일뿐** → 하방 리스크가 거의 발현 안 됨. 진짜 검증은 **약세장(2022) 포함 기간**에서 해야 함.

3. **단리(라오어) vs 복리(언이) 트레이드오프.** 복리는 실현이익을 재투입해 수익↑(SOXL +117%→+222%)이지만 **원금 초과 투입**(SOXL 최대 $18k)·**MDD↑**(25.8%→41.7%). 공격성↑.

4. **체결 모델은 일봉 근사.** LOC=종가 조건 체결, 지정가=장중 고가 도달 시 체결. 실제 분/틱 체결과 오차 가능.

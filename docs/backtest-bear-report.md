## 해석

### 1. 대폭락 방어력 = 압도적 (전략의 존재 이유)
- **2022년**: TQQQ B&H −74% → 전략 −8~−10%. SOXL B&H −81% → **+5~+8% 흑자**.
- V반등(코로나 SOXL): B&H −13% → **+32~36%**.
- 즉 폭락+반등 구간에서 무한매수가 진가 발휘. 단순보유 대비 손실 1/8 수준으로 압축.

### 2. 최악 = "천장 진입" 후 장기 하락 (고점→바닥)
- 라오어 TQQQ: **리버스 265일, 최저 −69%** — B&H(−49%)보다 *일시적으로 더 깊이* 물림. 평가금 $3,100까지 추락 후 −13%로 회복(사이클 0 = 기간 끝까지 미청산 보유).
- 천장에서 단일종목 전액 올인 = 무한매수도 오래·깊게 물린다. **죽진 않지만(원금 0 안 됨) 회복까지 1년+ 멘탈 싸움.**
- 교훈: 진입 타이밍 분산, 종목 분산(TQQQ+SOXL), 분할수↑(40↑)로 완화 가능.

### 3. 라오어(단리) vs 언이(복리) — 일관된 승자 없음
- TQQQ 고점→바닥: **언이 우위**(−23% vs 라오어 −13%? → 라오어가 수익은 나으나 trough −69로 더 깊음; 언이는 trough −53로 덜 깊음 = 언이가 낙폭 방어 우위).
- SOXL 고점→바닥: **라오어 압승**(+37% vs +4%).
- 복리는 회복장에서 수익↑, 하락장에서 낙폭↑(재투입으로 노출↑). 시나리오 의존. → C(둘 다 파라미터화)가 정답.

### 4. 공통: trough 30~70%는 여전히 발생
- 무한매수 = "안전"이 아니라 "**단순보유보다 덜 깨지고 빨리 회복**". 3x 레버리지는 어떤 운용을 해도 중간 낙폭 30%+는 일상.
- 리버스 모드는 손절 구간 — 평가손 확정하며 버티는 단계. 멘탈/규칙 준수가 실전 관건.
